## 标题
| 属性     | 作用               | 说明                                                                                             |
| -------- | ------------------ | ------------------------------------------------------------------------------------------------ |
| category | 路由匹配对比值     | 最先去找文件中的设置值 之后会设置为父级文件夹名称（中文不行） 或则pathF 与父级匹配不上会没有菜单 |
| subtitle | 菜单后的灰色小字体 |                                                                                                  |
| type     | 分组               | 默认分组文件夹名称                                                                               |
| title    | 菜单标题           | 默认文件名                                                                                       |
| path     | 自定义路由         | 默认文件名 中文会影像开发模式下刷新                                                              |
